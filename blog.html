No. 01: What is the difference between Map, Filter, forEach and find ?

Ans:
Map: map() performs for each element of array and returns a new array.
Filter: filter() search in array and filter element as a new array according to condition.
forEach: forEach() performs for each element of array but doesn't return anything.
find: find() search in array and returns the first element meeting the condition.

No. 02: What is the difference between Local Storage, Session storage and cookies ?

Ans:
Local Storage: store data permanently.
Session Storage: store data only for one session.
Cookies: send data to server.
