No. 01: What is the difference between Map, Filter, forEach and find ?

Ans:
Map: map() performs for each element of array and returns a new array.
Filter: filter() search in array and filter element as a new array according to condition.
forEach: forEach() performs for each element of array but doesn't return anything.
find: find() search in array and returns the first element meeting the condition.

No. 02: What is the difference between Local Storage, Session storage and cookies ?

Ans:
Local Storage: store data permanently.
Session Storage: store data only for one session.
Cookies: send data to server.

No. 03: What is Event loop? How does it works?

Ans: Event loop is a mechanism that allows JS to perform asynchronous task. Js is single threaded programming language but event loop helps it to run in asynchronous way.
When a asynchronous function like fetch() , setTimeOut comes to call stack it sends them to web api, and there they wait for there task to be completed ,such as data load. Also the task is removed from call stack, it that time other tasks are executed. When the task in the web api is completed it is loaded into callback queue. When the call stack is empty, event loop put task from callback queue to call stack and the task is then executed.
